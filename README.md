# Generador-Contrase-de-Contraseñas-Seguras
Este proyecto es una aplicación que se desarrollo en Python que genera contraseñas criptográficamente seguras basándose en las preferencias que tenga el usuario.

## 🛠️ Funcionalidades e Implementación
* **Estructuras Condicionales:** Filtra y añade caracteres (mayúsculas, números, caracteres especiales) según la preferencia del usuario.
* **Estructuras Repetitivas:** Utiliza bucles para mantener el menú interactivo activo y para construir la cadena de la contraseña de forma iterativa.
* **Seguridad:** Utiliza la librería `secrets` de Python para garantizar aleatoriedad real y mitigar riesgos de predictibilidad.
